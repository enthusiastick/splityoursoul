Icon | Character | Text
--- | --- | ---

![Banshee](https://wiki.bloodontheclocktower.com/images/6/63/Icon_banshee.png) | **Banshee** | If the Demon kills you, all players learn this. From now on, you may nominate twice per day and vote twice per nomination.
